What is Turborepo?

Turborepo is a high-performance build system for JavaScript and TypeScript codebases. Vercel recently [rewrote Turborepo from Go to Rust](https://vercel.com/blog/turborepo-migration-go-rust)—worth reading for context on performance and architecture.
